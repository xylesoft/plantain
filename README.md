# Plantain
A Sensor and Command system.

# Responsability
To provide an API which allows the creation of a connection between a UI and many I/O interfaces, Sensor (port) being the Input and Commands (Adapter) managing the Output.

# Domain

- Sensors
 - DHT11 Temperature
 - DHT11 Humidty
 - YL69 Soil Sensor

- Triggers
 - WaterPump
