# Plantain
A Sensor and Command system.

# Responsability
To provide an API which allows the creation of a connection between a UI and many I/O interfaces, Sensor being the Input and Command managing the Output.

- TriggerInterface

 - ServoTrigger
 - DigitalTrigger
